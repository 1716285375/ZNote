# 更新日志

本文档记录项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-XX

### 新增
- 首次公开发布
- 多平台视频下载支持（通过 yt-dlp 支持 Bilibili 和其他平台）
- 批量下载功能，支持多个视频和播放列表
- 智能任务队列，支持并发下载
- 下载历史管理，使用 JSON 存储
- 现代化 UI，支持浅色/深色主题
- 完整的设置系统：
  - 可配置下载路径
  - 文件命名（前缀/后缀）
  - 线程数配置 (1-10)
  - 声音通知
  - 自动打开目录选项
- 配置和历史记录的持久化存储
- 下载完成时的声音通知
- 自动打开下载目录功能
- 应用程序图标和品牌标识

### 技术细节
- 使用 Qt 6.9.1 和 C++17 构建
- 基于 CMake 的构建系统
- 分层架构 (UI, Service, Core, Utils)
- 使用 QMutex 的线程安全操作
- 基于 JSON 的配置和历史记录存储
- 跨平台支持 (Windows, Linux, macOS)
